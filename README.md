# Программа, получающая информацию о вакансиях с платформы hh.ru в РФ 
## Сохраняет вакансии в файл и позволяет их добавлять, фильтровать и удалять.

### Реализовать класс HeadHunterApi в модуле headhunter_api.py

Класс наследуется от абстрактного класса BaseClassAPI, для работы с платформой hh.ru.
Класс умеет подключаться к API и получать вакансии.

### Создать класс Vacancy в модуле vacancy.py для работы с вакансиями.

В этом классе определить атрибуты, такие как название вакансии, наименование работодателя, 
ссылка на вакансию, зарплата от и до, требования к соискателю. 
Класс поддерживает методы сравнения вакансий между собой по зарплате и 
валидирует данные (указана или нет зарплата), которыми инициализируются его атрибуты.

### Создать класс JSONSaver в модуле json_saver.py для сохранения информации о вакансиях в JSON-файл.

Класс JSONSaver наследуется от абстрактного класса BaseJSONSaver, который обязывает 
и реализованы методы для добавления вакансий в файл, получения данных из файла по указанным критериям и удаления информации о вакансиях. 
Класс JSONSaver сохраняет информацию о вакансиях в JSON-файл.

### Создать функцию user_interaction в модуле views.py для взаимодействия с пользователем.

Функция взаимодействует с пользователем через консоль. 
Возможности этой функции позволяют следующие критерии:
- ввести поисковый запрос для запроса вакансий из hh.ru;
- получить топ N вакансий по зарплате (N запрашивать у пользователя);
- получить вакансии с ключевым словом в описании.

В функции объединены все классы и функции в единую программу.
